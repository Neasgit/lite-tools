02 · Skill reference

All 14 skills in pipeline order. One job each.

01 · Discovery research

Searches news archives, court records, obituaries, and footage sources. Drops a structured Research Brief into the case folder from a single seed link.

Triggers · research case X fill the research brief kick off research start research
02 · Verification factcheck

Audits a Research Brief for accuracy, source quality, contradictions, and unsupported claims. Treats every claim as suspect until cross-sourced. Always run before scripting.

Triggers · verify case X fact-check this is this confirmed can we prove
03A · Open Records foia

Open Records / FOIA request drafter. Reads case Sources.md and Research Brief, identifies records not yet public, drafts formal requests with correct agency, statute, and retention-window language. Per-state statute lookup built in.

Triggers · draft a FOIA open records request request the bodycam what should I request
03B · Footage footage-fetch

Generates yt-dlp commands for footage URLs in the Research Brief, applies LITE file-naming convention, and builds the Footage Log scaffold. Does not download itself — produces paste-ready terminal commands.

Triggers · download footage for case X yt-dlp commands scaffold the footage log get the bodycam
04 · Script scriptwriter

Footage-led documentary script writing. Reads Research Brief and Footage Log first — never guesses at what footage shows. Uses the EWU voice rule: evidence drives the story, narration names what footage carries.

Triggers · write the script for case X draft the narration script this up
05 · 09B · Script Lock & Publish Gate qa

Script copy-editor and publish gate. Checks grammar, naming consistency, pronoun clarity, sentence variety, and voice adherence. Runs twice: Stage 5 before script lock, Stage 9B before publishing. Read-only — produces a QA Report.

Triggers · QA the script copy-edit case X ready to lock check script for issues proofread case X
06 · VO Prep vo-prep

Bridges script lock and ElevenLabs generation. Strips production markers, extracts pronunciation dictionary entries, and flags clone-friendliness issues. Outputs VO Text - ElevenLabs.md and VO Prep Report.md.

Triggers · prep the VO VO prep format for ElevenLabs extract pronunciations
08 · Long-Form Edit edit

Step-by-step walkthroughs for all video production questions. Covers Premiere Pro (primary NLE), CapCut (captions + yellow accent words), and OpusClip (Shorts). Runs at Stage 8 for the long-form and again at Stage 12 for Shorts.

Triggers · Premiere CapCut captions export B-roll audio sync timeline
09A · Legal legal-review

Pre-publish defamation and libel scan. Flags presumption-of-guilt language, unsupported claims about living people, civil-vs-criminal confusion, minor identification, and motive speculation. A structured self-audit — not a substitute for a real lawyer.

Triggers · legal review defamation check libel check is this safe to publish pre-publish legal
10 · Publish Prep package

Generates all publish assets in one pass: 3 title variants (per SEO formula), YouTube description, tags, pinned comment, and thumbnail brief. Reads locked script, Sources.md, and Footage Log.

Triggers · package case X publish prep generate title write description ready to publish
12 · Shorts shorts

Writes 3–5 vertical (9:16) Short scripts from a published long-form. Hook framework (counter-intuitive / question / footage / left-out), drip schedule across 3–4 weeks, EWU-style captions, reuses ElevenLabs narration.

Triggers · shorts for case X cut this into shorts shorts drip TikTok hooks vertical clips
13 · Community cut-notes

Drafts the YouTube Community tab post explaining what was cut from the long-form and why. 2–4 short paragraphs in the channel voice. Earns the "Every cut, on the record" tagline.

Triggers · cut notes for case X community post what was left out post transparency post
Async · Maintenance vault-mod

Read-only vault health check. Produces a dated Vault Audit file listing critical / minor / suggestions. Never writes — waits for approval. Runs outside the case pipeline on its own cadence.

Triggers · audit the vault is anything stale check for duplicates vault health check
Async · Prose stop-slop

Eliminates throat-clearing openers, binary-contrast clichés, false agency, em dashes, and other AI patterns from prose. Run before publishing scripts, descriptions, or community posts.

Triggers · slop this remove AI tells tighten the prose

04 · When skills overlap

Two skills can look like they do the same job. They don't.

research vs factcheck

research assembles the universe of claims. factcheck assumes the universe and stress-tests it. Never use research to verify; never use factcheck to discover.

foia vs footage-fetch

foia targets records that aren't public yet — it writes letters. footage-fetch targets URLs that already exist — it writes yt-dlp commands. Run both in parallel after factcheck.

qa runs twice

Once before script lock (Stage 5). Once before publish (Stage 9). The editorial errors that hurt are not the same as the script errors. Don't skip the second one.

legal-review vs qa

qa checks production readiness and editorial quality. legal-review checks for defamation and libel risk. Both run pre-publish. Different checks, same timing.

edit owns two stages

The long-form cut (Stage 8) and the Shorts cuts (Stage 12). Same skill, different priors. Don't try to do Shorts in the long-form session — context-switch costs.

vault-mod is async

Never blocks the case pipeline. Runs on its own cadence. If it's asking for input mid-case, defer it.
