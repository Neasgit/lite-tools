Tooling note

This page needs a local server.

Thumbnail Layouts is a design exploration page that compares six UI concepts for the brand reference. It loads its components as external .jsx files, which browsers block when the page is opened directly via file://.

To run it, start a tiny HTTP server from the site/ folder:

cd "C:\Users\Sean\YouTube\03 - Assets\Tools\site"
python -m http.server 8000

Then open http://localhost:8000/thumbnail-layouts.html.

The other tools in this folder — Brand Reference, generators, banners, expenses, etc. — all work from file:// directly.
